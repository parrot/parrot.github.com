NAME

src/pmc/fixedbooleanarray.pmc - fixed size array for booleans only

DESCRIPTION

The FixedBooleanArray PMC implements an array of fixed size, which stores booleans. It uses the Boolean PMC for all conversions. The FixedBooleanArray PMC is extended by the ResizableBooleanArray PMC.
