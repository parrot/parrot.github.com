NAME

src/pmc/orderedhash.pmc - Ordered Hash

DESCRIPTION

OrderedHash extends Hash to provide the interfaces of array and hash. To achieve ordering, hash there are a few restrictions: delete_keyed never removes items; they are just nulled.

Please note that if values are set via integer idx, these indices have to be in strict order. Using push_xx simplifies this task. This creates a key "\1idx" for idx and is therefore not fully transparent.

There are two iterator interfaces:

  • retrieve values (in creation order)

    • Please note that after a delete_keyed operation, iterating over values doesn't work any more. You'll get an error 'No such key'.

  • retrieve keys (in creation order)

See t/pmc/orderedhash.t.
