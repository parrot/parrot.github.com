NAME

t/debug.t - Lua Debug Library

SYNOPSIS 

    % perl t/debug.t

DESCRIPTION

Tests Lua Debug Library (implemented in languages/lua/src/lib/luadebug.pir).

See "Lua 5.1 Reference Manual", section 5.9 "The Debug Library", http://www.lua.org/manual/5.1/manual.html#5.9.

See "Programming in Lua", section 23 "The Debug Library".
