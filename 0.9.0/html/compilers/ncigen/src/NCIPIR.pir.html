NAME

NCIPIR::Compiler - NCI PIR Compiler for NCIGENAST trees.

DESCRIPTION

NCIPIR::Compiler defines a compiler that converts a NCIGENAST tree into PIR

METHODS

code([str])
Get/set the code generated by this compiler.
pir_children(node)
Return generated PIR for node and all of its children.
pir(Any node)
Return generated pir for any POST::Node. Returns the generated pir of node's children.
pir(POST::Op node)
Return pir for an operation node.
pir(POST::Label node)
Generate a label.
pir(POST::Label node)
Generate a label.
pir(POST::Label node)
Generate a label.
