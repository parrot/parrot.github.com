NAME

src/pmc/resizablebooleanarray.pmc - resizable array for booleans only

DESCRIPTION

The ResizableBooleanArray PMC implements an array of resizable size, which stores booleans. It uses the Boolean PMC for all conversions. The ResizableBooleanArray PMC extends the FixedBooleanArray PMC.
