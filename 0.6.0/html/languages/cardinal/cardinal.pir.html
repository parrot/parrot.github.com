TITLE

cardinal.pir - A cardinal compiler.

Description

This is the base file for the cardinal compiler.

This file includes the parsing and grammar rules from the src/ directory, loads the relevant PGE libraries, and registers the compiler under the name 'cardinal'.

Functions

onload()

Creates the cardinal compiler using a PCT::HLLCompiler object.

main(args :slurpy) :main

Start compilation by passing any command line args to the cardinal compiler.
