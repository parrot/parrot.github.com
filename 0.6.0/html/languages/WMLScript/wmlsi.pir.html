NAME

wmlsi - WMLScript Interpreter

SYNOPSYS 

 parrot wmlsi.pir file.wmlsc function [params ...]

DESCRIPTION

wmlsi translates a WMLScript bytecode file to Parrot PBC and calls function(params, ...).
