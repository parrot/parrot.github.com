NAME

examples/assembly/ncurses_life.imc - Conway's Life (ncurses version)

SYNOPSIS 

    % ./parrot examples/assembly/ncurses_life.imc examples/assembly/acorn.l

DESCRIPTION

An ncurses version of Conway's Life. acorn.l is a life file.

This version can load life.l files (#P, #A, #R formats only). You get a lot of .l files by installing xlife.

COMMANDS

q, Q

Quit.

8, 2, UP, DOWN

Move world up or down.

4, 6, LEFT, RIGHT

Move world left or right

5, HOME

Center world.

g

Toggle running the game.

o

Single step one generation.

s

Run slower.

f

Run faster.
