NAME

classes/closure.pmc - Closure PMC

DESCRIPTION

This class implements a closure, i.e. a subroutine which take a context structure.

Methods

void init()

Initializes the closure.

void mark()

Marks the closure as live.

void *invoke(void *next)

Invokes the closure.
