NAME

Parser - Perl extension for Miniperl parser

SYNOPSIS 

  use Parser;
  my $parser = Parser->new([$token_ref]);
  my $tree = $parser->parse([$token_ref]);

DESCRIPTION

Either new() or parse() accepts a $token_ref, but of course at least one has to.
