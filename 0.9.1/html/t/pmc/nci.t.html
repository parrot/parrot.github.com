NAME

t/pmc/nci.t - test the Native Call Interface

SYNOPSIS

    % prove t/pmc/nci.t

DESCRIPTION

This tests the Native Call Interface, that is the ParrotLibrary PMC. Most tests are skipped when the libnci_test.so shared library is not found.
