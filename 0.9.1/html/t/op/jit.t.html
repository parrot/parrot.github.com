NAME

t/op/jit.t - JIT register allocation

SYNOPSIS

        % prove t/op/jit.t

DESCRIPTION

Tests JIT register allocation. The tests are written for 4 mapped registers per kind, i.e. the crippled x86 architecture. If you are experimenting with register allocation please just use settings like in jit/i386/jit_emit, i.e. 4 mapped regs, 2 volatile ints ...

These tests are of course usable for other run cores too, the results should just be the same.
