NAME

config/gen/languages.pm - Build files for language implementations

DESCRIPTION

Config step for languages. Builds languages/Makefile and loops over list of languages. Special support for languages/dotnet.

A space separated list of languages can be passed in with the option 'languages'. An empty list of languages is OK. This means that only languages/Makefile will be regenerated.
