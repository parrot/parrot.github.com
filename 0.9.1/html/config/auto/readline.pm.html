NAME

config/auto/readline.pm - Probe for readline library

DESCRIPTION

Determines whether the platform supports readline. The GNU Project describes its version of the readline library as providing "... a set of functions for use by applications that allow users to edit command lines as they are typed in" (http://directory.fsf.org/project/readline/). Other readline libraries are, however, available and usable with Parrot.
