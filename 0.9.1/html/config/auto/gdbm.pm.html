NAME

config/auto/gdbm.pm - Test for GNU dbm (gdbm) library

DESCRIPTION

Determines whether the platform supports gdbm. This is needed for the dynamic GDBMHash PMC.

From http://www.gnu.org/software/gdbm/: "GNU dbm is a set of database routines that use extensible hashing. It works similar to the standard Unix dbm routines."
