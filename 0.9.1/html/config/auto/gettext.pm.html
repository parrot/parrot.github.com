NAME

config/auto/gettext.pm - Test for GNU native language support (gettext) library

DESCRIPTION

Determines whether the platform supports gettext. This is needed for parrot internationalization.

From http://www.gnu.org/software/gettext/: "[T]he GNU `gettext' utilities are a set of tools that provides a framework to help other GNU packages produce multi-lingual messages."
