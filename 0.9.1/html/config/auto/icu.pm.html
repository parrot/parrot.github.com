NAME

config/auto/icu.pm - Detect International Components for Unicode (ICU)

DESCRIPTION

Determines whether ICU is available. If so, configures ICU and adds appropriate targets to the Makefile.

From the ICU home page (http://www.icu-project.org/): "ICU is a mature, widely used set of C/C++ and Java libraries providing Unicode and Globalization support for software applications."
