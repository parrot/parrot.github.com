NAME

config/auto/jit - JIT Capability

DESCRIPTION

Determines whether there is JIT capability available. Use the --jitcapable and --execcapable options to override the default value calculated specifically for your CPU architecture and operating system.

Code formerly found in this step class used to determine characteristics of the CPU has been moved into the preceding step class, auto::arch.
