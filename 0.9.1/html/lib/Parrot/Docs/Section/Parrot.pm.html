NAME

Parrot::Docs::Section::Parrot - Root documentation section

SYNOPSIS

    use Parrot::Docs::Section::Parrot;

    Parrot::Docs::Section::Parrot->new->write_html;

DESCRIPTION

This is the top-level section for the HTML documentation for Parrot.

Class Methods

new()
Returns a new section.

Instance Methods

write_docs($silent, $delete)
Writes the HTML documentation.If $silent is true then progress is not reported.If $delete is true then the contents of $target will be deleted before any HTML is written. This argument is intended to be used only once by the root section, it is not passed to subsections.
