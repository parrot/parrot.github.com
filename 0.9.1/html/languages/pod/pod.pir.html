TITLE

pod.pir - A Pod compiler.

SYNOPSIS

as a command line (without interactive mode) :

    $ parrot pod.pbc document.text
    $ parrot pod.pbc --target=parse document.text
                                   PAST
                                   HTML

or as a library :

     load_bytecode 'pod.pbc'
     $P0 = compreg 'pod
     $S0 = <<'POD'
 Title
 =====
 Some text.
 POD
     $P1 = $P0.'compile'($S0)
     $S0 = $P1()
     print $S0

DESCRIPTION

This is the base file for the Pod compiler.

Functions

onload()
Creates the Pod compiler using a PCT::HLLCompiler object.
html(source [, adverbs :slurpy :named])
Transform Pod AST source into a String containing HTML.
main(args :slurpy) :main
Start compilation by passing any command line args to the Pod compiler.
