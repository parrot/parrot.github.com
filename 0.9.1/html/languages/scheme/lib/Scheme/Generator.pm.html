NAME

Scheme::Generator - The Scheme code generator

SYNOPSIS

  use Scheme:Generator;

  my @code = Scheme::Generator->new($code_tree)->generate();

DESCRIPTION

The code generator reads in a tree structure, and walks that to generate the output.
