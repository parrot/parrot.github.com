NAME

Scheme - an implementation of Scheme on Parrot

Description

Compile Scheme to PASM.

Implementation

Lexing is done in Scheme::Tokenizer. Parsing is done Scheme::Parser, PIR is generated in Scheme::Generator. Scheme.pm pulls it all together.
