NAME

examples/io/net_smtp.pasm - SMTP Example

SYNOPSIS

    % ./parrot examples/io/net_smtp.pasm

DESCRIPTION

A sample of the network ops. Connects to SMTP (port 25). Doesn't work very well but at least shows the idea.

You should be running the echo service on your box (port 7). Be sure to set PARROT_NET_DEVEL to 1 in io/io_private.h and rebuld Parrot or the network layer won't exist.
