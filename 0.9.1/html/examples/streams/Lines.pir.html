INFORMATION

This example demonstrates what Stream::Lines does.

FUNCTIONS

_main
Creates a stream and pipes it through a Stream::Lines stream.
_text
This sub is used as a source for a stream piped through Stream::Lines. Writes some text containing newlines to the stream.Stream::Lines will return it one line per read.
