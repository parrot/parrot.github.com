INFORMATION

This small example shows the usage of Stream::ParrotIO.

It reads this file with a default block size.

You can specify another block size with the blockSize method. blockSize called without an integer parameter will return the current block size.

Each time the read method is called, the next block is read from the underlying ParrotIO, until EOF, where the stream will be disconnected.

Instead of using the open method, you can also assign your own ParrotIO PMC to the stream with the assign op.
