NAME

examples/benchmarks/gc_header_new.pasm - GC Benchmark

SYNOPSIS

    % ./parrot examples/benchmarks/gc_header_new.pasm

DESCRIPTION

Makes some empty strings with concat, then prints out some statistics indicating:

  • the time taken
  • the total number of bytes allocated
  • the total of GC runs made
  • the total number of collection runs made
  • the total number of bytes copied
  • the number of active Buffer structs
  • the total number of Buffer structs
