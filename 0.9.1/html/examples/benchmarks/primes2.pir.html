NAME

examples/benchmarks/primes2.pir - Calculate prime numbers < 5000

SYNOPSIS

    % time ./parrot examples/benchmarks/primes2.pir

DESCRIPTION

Calculates all the prime numbers up to 5000 and prints out the number of primes and the last one found. Use PMCs, not ints.
