TITLE

Stream::Writer - a PIR sub as target for a Stream

VERSION

version 0.1

SYNOPSIS

    # create the stream
    new stream, "Stream::Writer"

    # set the source sub
    .const 'Sub' temp = "_reader"
    stream."source"( temp )

    stream."write"( "hello, world" )

    ...

    .sub _reader :method
        .local string str
        str = self."read"()
    .end

DESCRIPTION

METHODS

source."write"()
...
source."rawRead"() (internal)
...
