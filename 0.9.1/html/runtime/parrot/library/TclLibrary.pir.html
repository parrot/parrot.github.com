_load_lib_with_fallbacks(string friendly_name, pmc fallback_list)
This function is more generally useful than just for this module -- it implements the search for a particular libary that may appear under any of several different filenames. The fallback_list should be a simple array of strings, each naming one of the possible filenames, without the trailing shared library extension (e.g. .dll or .so).
