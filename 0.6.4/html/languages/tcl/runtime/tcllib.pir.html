tcllib

This file contains all the PIR necessary to setup the basic Tcl, _Tcl, and _TclWord namespaces. These namespaces can then be used by our own tcl.pir to setup a tclsh-like interpreter, or to allow other PIR programs to access procedures in our own namespaces, also providing a compreg-compatible method.
