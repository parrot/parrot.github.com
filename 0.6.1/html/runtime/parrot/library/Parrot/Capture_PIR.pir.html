NAME

Capture_PIR - PIR-based Capture class

DESCRIPTION

This file implements a Capture class of objects. Ideally we'd like to use the Capture class that is written in C ("pmc/capture.pmc" in src), but unfortunately Parrot currently has a limitation in handling subclasses of the built-in Capture. So, this is a temporary PIR-based version until the Capture PMC is working properly.
