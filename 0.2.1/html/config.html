parrotcode: Configuration
Contents

Parrot is configured by running the Configure.pl script. This is essentially just a wrapper around Parrot::Configure::RunSteps. The steps are listed below in the order in which they are performed.

Configure.pl
Parrot's Configuration Script.

config/init/manifest.pl
MANIFEST Check.

Configuration Defaults.

config/init/hints.pl
Platform Hints.

config/init/headers.pl
Nongenerated Headers.

config/init/optimize.pl
Optimization.
