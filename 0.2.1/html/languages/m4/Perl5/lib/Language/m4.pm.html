NAME

Language::m4 - A port of GNU m4 from C to Perl5

SYNOPSIS 

  use Language::m4;

  Language::m4::main();

DESCRIPTION

Provides a main function for the Perl scipt m4.

USAGE

Uses the parameters from the command line.
