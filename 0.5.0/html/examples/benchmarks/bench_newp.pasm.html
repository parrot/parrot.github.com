NAME

examples/benchmarks/bench_new.pasm - PMC Creation

SYNOPSIS 

    % time ./parrot examples/benchmarks/bench_new.pasm

DESCRIPTION

Creates a ResizablePMCArray PMC and fills it with Integer PMCs. Then prints out some statistics indicating:

  • the time taken

  • the total number of bytes allocated

  • the total of DOD runs made

  • the total number of collection runs made

  • the total number of bytes copied

  • the number of active Buffer structs

  • the total number of Buffer structs
