find_named_param

Look up a named parameter by name in a signature, and return the P6C::sigparam object if found. This only looks at named parameters, not positional params.

find_positional_param_by_name

Look up a positional parameter by name in a signature, and return the P6C::sigparam object if found. This only looks at positional parameters, not named parameters. (Remember, positional params also have names.)

categorize_args

Split up an array of args into positional params, named params whose names are statically known, named parameters whose names are unknown at compile time, etc.
