NAME

examples/benchmarks/addit.pasm - Variable Argument Subroutines

SYNOPSIS 

    % time ./parrot examples/benchmarks/addit.pasm

DESCRIPTION

Joe Wilson's original PASM version of his varargs subroutines benchmark. It calls an "add it" function 500000 times in a loop and prints the result (21001097.970000).
