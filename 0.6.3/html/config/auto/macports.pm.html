NAME

config/auto/macports.pm - Determine Macports location on Darwin

DESCRIPTION

If the operating system is Darwin, this class determines whether or not Macports is installed in the default location.

Note: No provision is yet made for Macports installation in non-default locations.

Note: This step is currently Darwin/Macports-specific because it consolidates code previously found in multiple configuration step classes. With some modification it may be suitable for application to BSD ports programs in general, but as there was no BSD-specific code in the configuration step classes, that modification has not yet been made.
