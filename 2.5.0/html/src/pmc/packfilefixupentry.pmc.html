NAME

src/pmc/packfilefixupentry.pmc - PackfileFixupEntry PMC

DESCRIPTION

This class implements a PackfileFixupEntry object. It is an element in the PackfileFixupTable segment of the .pbc file. See PDD13 for a design spec.
