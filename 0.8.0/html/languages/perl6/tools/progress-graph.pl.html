NAME

progress-graph.pl - generate a chart that displays rakudo's progress with passing tests.

SYNOPSIS

perl tools/progress-graph.pl [input_file [output_file]]

DESCRIPTION

Create a chart that displays the number of passed, skipped and TODO tests.

If input_file is given at the command line, it is read and parsed as a CSV file. if absent docs/spectest-progress.csv is used.

If output_file is given, the image is written to that file. If not, the file name rakudo-tests-%s.png is used, where %s stands for the last processed date.
