NAME

examples/mops/mops.c - Calculate M ops/s

SYNOPSIS 

    % make examples/mops/mops
    % time examples/mops/mops

DESCRIPTION

A Perl 6 implementation of the examples/mops/mops.pasm example program, for speed comparisons.

Calculates a value for M ops/s (million operations per second) using integer arithmetic.

Prints out:

  • the number of look iterations,

  • the estimated number of ops performed,

  • the elapsed time, and

  • the number of M op/s.
