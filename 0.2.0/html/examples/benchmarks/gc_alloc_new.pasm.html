NAME

examples/benchmarks/gc_alloc_new.pasm - GC/DOD Benchmark

SYNOPSIS 

    % ./parrot examples/benchmarks/gc_alloc_new.pasm

DESCRIPTION

Makes a big string with repeat, then prints out some statistics indicating:

  • the time taken

  • the total number of bytes allocated

  • the total of DOD runs made

  • the total number of collection runs made

  • the total number of bytes copied

  • the number of active Buffer structs

  • the total number of Buffer structs
